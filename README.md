# Infobackground — лендинг студии сайтов

Премиальный минималистичный лендинг о создании сайтов и лендингов для бизнеса
и частных предпринимателей. Один статический файл — без сборки и зависимостей.

## Особенности

- **Три языка**: русский, английский, украинский (переключатель в шапке,
  выбор запоминается в `localStorage`, автоопределение по языку браузера)
- **Видео-фон в hero**: зацикленный пинг-понг ролик (`assets/hero-loop.webm` /
  `.mp4`), постер-фолбэк и уважение к `prefers-reduced-motion`
- **Палитра** из брендового референса: тёмный тил `#071518`, янтарный `#F6A81F`,
  оранжевый `#FF5C1E`, красный акцент `#C2261F`
- **Форма заявки** отправляется в n8n-webhook (тот же, что у основного сайта)
- Прелоадер, кастомный курсор, reveal-анимации, счётчики, маркиза стека,
  FAQ-аккордеон — всё на ванильном JS, без библиотек

## Запуск

Любой статический хостинг (GitHub Pages, Netlify, Vercel) или локально:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Структура

```
index.html            — страница целиком (стили, разметка, i18n, скрипты)
assets/hero-loop.webm — фоновое видео hero (VP9, пинг-понг)
assets/hero-loop.mp4  — фоновое видео hero (H.264, фолбэк)
assets/hero-poster.jpg— постер видео
assets/founder.webp   — фото основателя
```
